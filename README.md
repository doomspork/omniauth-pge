# OmniAuth PG&E

OmniAuth strategy for [PG&E's Share My Data](http://www.pge.com/en/myhome/addservices/sharemydata/vendor/testrequirements/index.page) using OAuth2.

## Installation

    $ gem install omniauth-pge
